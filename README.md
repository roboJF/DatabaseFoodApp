# DatabaseFoodApp

Foodie
By: Raphael Zeldin, Jaden Fiero, Geena Sarnoski, Jack Tinik

Install instructions:  
Download MySQL Server, MySQL Workbench and Jconnector, all for the same version of SQL  
Run schema.sql in MySQL Workbench  
Create a lib folder in the project directory, and add Jconnector as an external library  


Run Instructions:
First run schema.sql and sample_data.sql in MySQL Workbench, in that order
In the terminal, run the following command based on the java version you are running
For Java 17: java -jar fooddelivery_17.jar
For Java 24: java -jar fooddelivery_24.jar
Log into MySQL by using the username root and whatever password you set
Have Fun!
